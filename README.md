## Overview

**HGP** is a framework for on-device personalized agent memory using hybrid graph storage. It enables AI agents to maintain efficient, structured, and personalized long-term memory directly on-device, combining the strengths of both property graphs and vector-based retrieval to support complex reasoning and personalization.

### Key Features

- **Hybrid Graph Storage**: Integrates structured property graphs with dense vector representations for rich, multi-relational memory.
- **On-Device Personalization**: Memory is maintained and updated locally, enabling privacy-preserving personalized agent behavior.
- **Efficient Retrieval**: Supports fast and semantically meaningful memory retrieval for real-time agent interactions.
- **Scalable Design**: Designed to work efficiently within the resource constraints of on-device deployments.
